# 经典游戏合集 - 规格说明书

## 1. 项目概述
- **项目名称**: Classic Games Collection
- **类型**: 经典街机游戏合集
- **核心功能**: 包含多个经典游戏的游戏面板，可选择并进入不同游戏
- **目标用户**: 所有年龄段玩家

## 2. 游戏列表
1. **贪吃蛇 (Snake)** - 经典贪吃蛇游戏
2. **俄罗斯方块 (Tetris)** - 经典俄罗斯方块游戏

## 3. 视觉与渲染规格

### 整体风格
- **主题**: 复古街机/霓虹风格
- **字体**: "Press Start 2P" (像素风格)
- **主色调**: 深色背景 + 霓虹绿/紫色点缀
- **背景**: #0a0a1a (深空黑)

### 颜色方案
- 主色: #00ff88 (霓虹绿)
- 次色: #9d4edd (霓虹紫)
- 强调色: #ff4757 (红色)
- 背景: #0a0a1a
- 卡片背景: #1a1a2e
- 文字: #ffffff

### 页面布局
- **首页**: 游戏卡片网格布局，每个游戏一张卡片
- **游戏页面**: 全屏游戏区域 + 顶部分数/控制信息

## 4. 首页 - 游戏面板

### 结构
- 顶部标题 "CLASSIC GAMES"
- 游戏卡片网格 (自适应屏幕)
- 每个卡片包含: 游戏图标/预览、游戏名称

### 游戏卡片
- 尺寸: 280x320px
- 圆角: 16px
- 悬停效果: 发光边框 + 轻微上浮
- 点击跳转至对应游戏

## 5. 贪吃蛇游戏规格

### 画布
- 尺寸: 600x600 像素
- 网格: 20x20

### 视觉
- 蛇: 霓虹绿圆角矩形，头部带阴影
- 食物: 红色脉冲圆形
- 背景: 深色 + 细微网格线

### 机制
- 速度: 150ms/格
- 计分: 每吃一个食物 +10 分
- 最高分持久化 (localStorage)

### 控制
- 方向键: 移动
- 空格: 暂停
- Enter: 开始/重玩

## 6. 俄罗斯方块游戏规格

### 画布
- 尺寸: 400x600 像素
- 网格: 10x20 (每格 30px)

### 方块类型 (Tetrominoes)
- I型: 青色 (00ffff)
- O型: 黄色 (ffff00)
- T型: 紫色 (9d4edd)
- S型: 绿色 (00ff00)
- Z型: 红色 (ff0000)
- J型: 蓝色 (0000ff)
- L型: 橙色 (ff8800)

### 视觉
- 方块: 带圆角和边框
- 已固定方块: 半透明版本
- 下一个方块预览: 右侧面板

### 机制
- 速度: 初始 500ms，下降加速
- 计分:
  - 消除1行: 100分
  - 消除2行: 300分
  - 消除3行: 500分
  - 消除4行: 800分
- 最高分持久化 (localStorage)
- 下一个方块预览

### 控制
- 方向左/右: 移动
- 方向下: 加速下落
- 方向上: 旋转
- 空格: 暂停
- Enter: 开始/重玩

## 7. 交互规格

### 导航
- 首页点击游戏卡片进入游戏
- 游戏内提供"返回首页"按钮

### 共享UI
- 分数显示
- 最高分显示
- 暂停/继续功能
- 游戏结束界面

## 8. 技术实现
- 纯 HTML5 + CSS3 + JavaScript
- Canvas 2D 渲染
- 无外部依赖 (除 Google Fonts)
- localStorage 持久化

## 9. 验收标准
- [ ] 首页正确显示所有游戏卡片
- [ ] 点击卡片进入对应游戏
- [ ] 每个游戏完整可玩
- [ ] 分数和最高分正确记录
- [ ] 返回按钮可返回首页
- [ ] 视觉风格统一
- [ ] 控制响应正常
